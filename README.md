1. SOFTWARE OVERVIEW
This software is a next-generation digital blackboard (whiteboard) system 
designed intuitively for school classrooms, cram schools, online teaching, 
and student self-study. It features a built-in "Dynamic Geometry Engine" 
that mathematically connects and syncs points and lines.

Reflecting classroom feedback to maximize board space, Ver.38.0 introduces 
a new system to collapse the toolbar into a tiny icon at any of the four 
corners (Top-Left, Top-Right, Bottom-Left, Bottom-Right). This achieves 
a "Wide Canvas" completely free of writing obstructions.

2. KEY FEATURES
- Corner Hide/Expand System (New in Ver.38.0): 
  Collapse the toolbar into any corner and restore it instantly with one tap.
- 6 Dynamically Differentiated Drawing Styles: 
  Master Calligraphy Brush (speed/pressure reactive), Textured Chalk, etc.
- Real-time Geometric Syncing: 
  Vertex dragging automatically recalculates lengths, angles, orbits, and lines.
- Rich Educational Tools: 
  "Inscribed Angle Theorem" preset, 3 learning backgrounds (Grid, 4-Lines, 
  Square-Grid), Lasso grouping, JSON save/load, and instant JP/EN toggle.

3. INSTALLATION & LAUNCH
Unzip the downloaded file and double-click or drag-and-drop the HTML file 
into your browser. It runs instantly and works fully offline.
