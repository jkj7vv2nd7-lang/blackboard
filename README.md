アプリ詳細操作説明マニュアル（全機能解説）
本アプリは、直感的なアイコンと高度な数学的演算ロジックが融合したインターフェースを採用しています。以下に、全画面構成と機能の詳細を解説します。 
① 画面構成・インターフェースの基本
·	メインキャンバス（黒板）: 深みのある黒板色（#0d1f16）を基調とし、長時間の授業でも児童・生徒の目の疲労を抑えます。 
·	メインツールバー: 画面上部に浮遊するコントロールセンターです。左端の「⣿（ハンドル）」をドラッグすることで、画面内の邪魔にならない場所へ自由に移動できます。 
·	情報メーター（右下パネル）: 現在選択中の図形の長さ、角度、直線の数式（傾き・切片）などがリアルタイムに表示・更新されるデータセンターです。 
·	しきをつくろう！パレット（左下関数パネル）: 背景を座標平面に切り替えた際、数式の直接入力や、分数・2乗（$x^2$）の視覚的入力を行う数学専用のキーパッドです。 
② コマンド・ボタン詳細解説
【画面・管理グループ】
·	📥 閉じる / 格納機能: ツールバー左端の「閉じる」を押すとモーダルが開き、画面の四隅（左上・右上・左下・右下）のいずれかにツールバーを「🛠️」という小さなアイコンとして最小化格納できます。授業中に黒板を広く見せたいときに最適です。再度タップすると元のサイズに復元します。 
·	🔍 UIサイズ / 🔄 配置切替: 「UIサイズ」を押すたびに、ツールバー全体のサイズが【小・中・大】と切り替わり、児童のタッチ操作や教師の提示用など環境に合わせられます。「配置切替」を押すと、ツールバーが【横一列型】から【縦一列型】へトランスフォームし、画面側面に配置可能になります。 
·	💾 保存 / 📤 読込: 現在キャンバスに描かれている幾何データ、手書き線、測定状態のすべてを「.json」形式のファイルとしてローカルPCに保存・読み込みが可能です。前日の授業の続きをやりたい時や、教師が事前に自作した幾何教材を配布・提示する際に重宝します。 
【設定・描画カスタマイズグループ】
·	🏁 背景変更（5つのモード）: 押すたびに背景が切り替わります。 
1.	背景: なし（全面グリーン） 
2.	背景: 通常方眼（一般的な算数・数学用） 
3.	背景: 英語4線（外国語活動・英語の板書用。赤い3線目が基準） 
4.	背景: 十字マス（国語の漢字練習や低学年のノート指導用） 
5.	背景: 座標平面軸（$x$軸・$y$軸と目盛り、数字が表示され、関数モードが起動） 
·	カラーセレクター: 【白・赤・黄・青】の4色を搭載。チョークらしい発色を再現しています。 
·	描画スタイルの詳細設定:
o	ペン種類: 「🖍️チョーク（粉が散るリアルな質感）」「✏️鉛筆（やや薄く細い描画）」「🖌️毛筆（速度感が出る太さ変化）」「🖊️蛍光ペン（不透明度が下がり背後が透ける）」「✨ネオン（美しく発光するサイバーペン）」から選択可能。 
o	線種・太さ: 「━ 実線」と「--- 破線（補助線用）」の切り替え、および4段階の太さ調整（細・中・太・極太）に対応。 
【作図・図形教材ツールグループ】
·	✏️ 手書き: フリーハンドで文字やイラストを描けます。 
·	📍 点を打つ: タップした位置に幾何学的な結節点（アルファベットのラベル A, B, C... が自動で割り振られる点）を連続して配置します。 
·	📏 線分 / ➖ 直線 / 🪶 半直線 / ➔ ベクトル:
o	線分: 2点間を真っ直ぐ結ぶ有限の線（辺）。 
o	直線: 2点を通って無限に伸びる線。 
o	半直線: 始点から始まり、2点目の方向へ無限に伸びる光線。 
o	ベクトル: 向きと大きさを持った矢印の線。 
·	🎯◯ コンパス / 🌐 円弧:
o	コンパス: 始点（中心）を決めて外側にドラッグすると、本物のコンパスを模した美しい脚部・鉛筆アニメーションが駆動し、真円を描きます。 
o	円弧: 中心から半径を指定して、円の一部（扇形の外周）を描きます。「動かす変形」モードで端点をつまむことで、扇形の角度を広げたり狭めたりできます。 
·	⇄ 平行線 / ┼ 垂直線: 基準にしたい既存の直線を1回タップするとオレンジ色に点滅してロックされます。その後、配置したい場所へ向かってドラッグすると、基準線に対して完全に平行、または垂直な直線を引き出すことができます。 
·	🔺図形教材プリセット: 「一般の三角形」「二等辺三角形」「正三角形」「直角三角形」「正方形」「ひし形」「台形」「平行四辺形」を瞬時に画面中央に生成します。すべての頂点は個別ラベルを持ち、後述の「動かす」モードで自由に変形可能です。 
【探求操作・動的幾何学グループ（※一部は高度モードで起動）】
·	🖐️ 動かす変形: 本アプリの核心機能です。描画された「点」をドラッグすると、その点に繋がっている線分や円、ベクトルが幾何学的な連続性を維持したまま動的に変形します。 
·	🔒 点を固定（高度モード）: 点をタップするとその点が「赤くロック」され、他の点を動かしてもその位置から動かなくなります（図形の底辺を固定したい時に使用）。 
·	⛓️ 動きの通り道（高度モード）: 動かしたい点を選んだあと、コンパス等で描いた「円」をタップすると、その点は円周上のレールから外れなくなり、円軌道上のみをスムーズに周回移動するようになります（動点問題の再現に最適）。 
·	🎯 投げ縄選択 / 🔒 グループ化 / 🔓 解除（高度モード）: 画面を指で囲むことで、複数の点や線を一括選択し、それらを一つの「グループ」として固定できます。グループ化された図形は、形を崩さずに平行移動させることができます。 
·	🎡 円周角定理（スペシャル教材）: ワンタップで「円、中心角、円周角、動点P」からなる定理探求セットが自動構築されます。 
【測定・履歴・消去グループ】
·	📏 長さ測定: 描画された線分をタップするだけで、その2点間のデジタルな長さ（距離）をリアルタイム計測して黒板上および情報メーターに表示します。変形させると数値も連動します。 
·	📐 角度測定: 方法は2種類あります。①3つの頂点を「反時計回りまたは時計回り」に順番にタップする。②交わっている2つの線分をタップする。これにより、内角の角度（$^{\circ}$）が円弧のデザインと共に黒板へ表示されます。右下の情報メーターに表示される「🔄 反転（Flip）」ボタンを押すと、優角（外側の角）と劣角（内側の角）の計測を瞬時に切り替えることができます。 
·	↩️ 元に戻す / ↪️ やり直す: 最大40手前までの操作履歴を記憶しており、作図の失敗をいつでも取り消せます。 
·	🧼 消しゴム / ❌ 全消去: 消しゴムモードは、手書き線をなぞって消せるだけでなく、コンパスの円や線分を擦ることで、擦った部分を動的に分解・削除することができます。「全消去」は確認ダイアログを経て黒板を一瞬でリセットします。 
3. 各種授業での具体的な役立て方・活用アイデア集
本アプリは、その動的な性質を活かすことで、従来の静的な教科書や黒板では不可能だった「めあて・課題の発見」「予想・仮説の検証」「きまりの一般化」を子どもたち自身の手で実現できます。
① 小学校 算数での活用
【3年：三角形のなかま分けと特長】
·	活用法: 「図形教材」から「二等辺三角形」や「正三角形」を呼び出します。「動かす変形」モードで頂点をつまんで引っ張ると、全体の形や傾きは大きく変わりますが、「2つの辺の長さが等しい」「3つの辺の長さがすべて等しい」という性質を維持したまま変形します。 
·	効果: どんなに平べったい形に変形させても二等辺三角形は二等辺三角形のままであること（不変性）を視覚的に理解し、「2つの辺の長さが等しい図形」という定義の本質に気づかせることができます。
【5年：図形の角の大きさを調べよう（多角形の内角の和）】
·	活用法: プリセットや「点を打つ」機能で自由な四角形や五角形を作ります。「角度測定」ツールですべての内角を表示させます。情報メーターに表示された4つ（または5つ）の角度を子どもたちにノートで合計させます。その後、教師が図形の形をぐにゃぐにゃと大きく変形させます。 
·	効果: 各個別の角度は「$50^{\circ} \rightarrow 85^{\circ}$」などと激しく変化するにもかかわらず、「それらをすべて足すと、変形前も変形後も必ず360度（または540度）になる」という事実をリアルタイムに目の当たりにすることで、深い感動と共に定理の普遍性を学べます。
② 中学校・高校 数学での活用
【中学2年：平行線と角の性質 / 中学3年：円周角の定理】
·	活用法: 「🎡 円周角定理」ボタンを1タップして専用モデルを起動します。画面には最初から中心角と円周角の度数が測定された状態で円が表示されます。動かすモード（🖐️）を選択し、円周上の赤い点Pを掴んで、ぐるりと周回させてみせます。 
·	効果: 点Pをどこに移動させても、円周角の大きさが「全く変わらない（例：ずっと$35.5^{\circ}$）」こと、そして中心角の大きさ（例：$71.0^{\circ}$）の「常にピッタリ半分（2分の1）」になっていることがデジタルで証明されます。生徒に点Pを操作させることで、主体的な探求活動（GIGAスクール端末での個別最適な学び）へスムーズに移行できます。 
【中学3年：関数 $y=ax^2$ のグラフの特長】
·	活用法: 背景を「座標平面軸」に変更します（左下に「しきをつくろう！」パレットが出現）。子ども用テンキーや分数UIを使って、0.2 * x * x（$y=0.2x^2$）を入力したり、数式を書き換えます。また、描画した直線の傾きを「動かす」モードで変えると、情報メーター内の数式（$y=2x+1$ など）がリアルタイムに書き換わる様子を提示します。 
·	効果: 比例定数 $a$ の値を「分数UI」を使って「$1/2 \rightarrow 1/4 \rightarrow 1/8$」と小さくしていくと、グラフの開き方がどのように変化するかをアニメーションのように体感できます。「数式（抽象）」と「グラフ（具象）」の架け橋となる双方向学習が容易に達成されます。 
③ その他の教科（英語・国語・理科・総合）でのユニバーサルな活用
【外国語活動・英語：アルファベット指導（4線ノートの再現）】
·	活用法: 背景を「英語4線」モードに切り替えます。ペン種類を「🖍️チョーク」または「🖊️蛍光ペン」の太（12）に設定します。 
·	効果: 3本目の赤い線（ベースライン）を意識させながら、「小文字の g や p はどこまで下に突き抜けて書くべきか」を、教室の大型テレビに本物のノートと同じ見た目で美しく板書・指導できます。
【国語：漢字の書き順・バランス指導】
·	活用法: 背景を「十字マス」モードに変更します。ペンを「🖌️毛筆スタイル」にします。 
·	効果: マスの中心（十字の点線部分）からどこに筆を入れ、どこで「はね・はらい」を行うかを、習字の半紙を提示するようにダイナミックに実演指導できます。
【理科・総合：ベクトルの合成と力のつり合い・プログラミング学習】
·	活用法: 「➔ ベクトル」作図機能を用いて、1つの点から異なる2方向の矢印（力）を描きます。その後「平行線」ツールを用いてそれぞれの補助線を引き、平行四辺形を作って合成ベクトル（対角線）を視覚化します。 
·	効果: 力の大きさと向きが、矢印の長さと傾きで完全に数学的にシミュレートできるため、理科の「力の合成・分解」の単元を数学的アプローチで分かりやすく解説できます。
4. 総括：このアプリがもたらす教育的パラダイムシフト
この「最高峰デジタル学習黒板」は、従来の「先生が前で綺麗に描いて見せるための道具」を遥かに超越しています。 
1.	「なぜ？」「もし〜なら？」をその場で検証できる: 生徒から「先生、その三角形の角を右に引っ張ったらどうなるの？」という突発的な質問が出た瞬間、教師は「動かす変形」ツールで即座にその状態を作り出し、全員で数値を検証できます。 
2.	失敗を恐れない作図環境: コンパスの作図ミス（ズレ）や線の引き間違いも、「↩️元に戻す」や動的な「消しゴム分解」によって一瞬でリトライ可能です。これにより、作図が苦手な児童・生徒も意欲的に図形学習に取り組めるようになります。 
3.	教材のデジタル資産化: 作成した美しい問題モデルやグラフは、「💾保存」ボタンでJSONデータとして保管し、翌年の授業や他のクラスの先生方と一瞬で共有・再利用が可能です。授業準備（教材研究）の時間を劇的に削減し、校内でのチーム学校としての教材共有を強力に推進します。
