▼1. 概要
本ソフトは、学校の授業や家庭学習で「直感的に、美しく、深く学べる」ことを追求した超高性能なデジタル黒板アプリケーションです。
最新バージョンでは、起動時の画面をスッキリと「手書きモード」に統一し、必要なときだけ「数学モード」を呼び出せるハイブリッド仕様へと進化しました。チョークのリアルな質感の表現から、最先端の動的幾何学（動かす変形）、グラフの自動描画まで、これ一枚ですべて対応します。

▼2. 今回の主な修正・変更点
・【画面の最適化】起動時はツールバーをスッキリさせ、「手書き」を中心としたシンプルな画面にしました。
・【数学モード切り替え】「数学モードへ」ボタンを押すだけで、コンパスや各種作図、図形教材、動かす変形などの高度な機能がいつでもワンタッチで現れます。
・【システムグループの統合】管理、履歴、共通グループを「システム」として1つにまとめ、より操作しやすくなりました。
・【コンパスの改良】描画中のコンパスアニメーションをスッキリとさせ、より作図に集中できるようビジュアルを洗練しました。
・【背景機能の安定化】通常方眼、英語4線、国語の十字マス、座標平面軸への切り替えをより正確に動作するよう調整しました。

▼3. インストール・起動方法

ダウンロードした「blackboard.html」をダブルクリックするか、お使いのブラウザにドラッグ＆ドロップしてください。

インターネットに接続していなくても、完全にオフラインで動作します。
